
### Herramientas para instalar

Instale las siguientes herramientas:

- [Visual Studio](https://visualstudio.microsoft.com/) - Este es el entorno de desarrollo donde se producirá el proceso de codificación y compilación. Instale el tiempo de ejecución C/C.
    
- [x64dbg](https://x64dbg.com/) - x64dbg es un depurador que se utilizará en todos los módulos para obtener una comprensión interna del malware desarrollado.
    
- [PE-Bear](https://github.com/hasherezade/pe-bear) - PE-bear es una herramienta de reversión multiplataforma para archivos PE. También se utilizará para evaluar el malware desarrollado y buscar indicadores sospechosos.
    
- [Process Hacker 2](https://processhacker.sourceforge.io/downloads.php) - Process Hacker es una herramienta poderosa y polivalente que ayuda a monitorear los recursos del sistema, depurar software y detectar malware.
    
- [Msfvenom](https://www.offensive-security.com/metasploit-unleashed/msfvenom/) - Msfvenom es una herramienta de interfaz de línea de comandos que se utiliza para crear, manipular y descargas útiles de salida.
