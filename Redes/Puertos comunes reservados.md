
#puertos

|Nombre|Puerto/Protocolo|Descripción|
|---|---|---|
|Echo|7/TCP y 7/UDP|Prueba de conectividad de red, devuelve el mensaje recibido.|
|Discard|9/TCP y 9/UDP|Elimina los datos recibidos sin procesarlos.|
|Daytime|13/TCP y 13/UDP|Devuelve la hora actual en formato de texto.|
|QOTD|17/UDP|Devuelve una cita o frase al azar.|
|Chargen|19/TCP y 19/UDP|Genera y devuelve una cadena de caracteres al azar.|
|FTP-DATA|20/TCP|Protocolo de transferencia de datos de FTP, utilizado para transferir archivos entre computadoras.|
|FTP|21/TCP|Protocolo de transferencia de archivos, utilizado para transferir archivos entre computadoras.|
|SSH|22/TCP|Protocolo de shell seguro, utilizado para acceder a una computadora remota de forma segura.|
|Telnet|23/TCP|Protocolo de terminal remota, utilizado para acceder a una computadora remota de forma insegura.|
|SMTP|25/TCP|Protocolo de transferencia de correo simple, utilizado para enviar correos electrónicos.|
|Time|37/UDP|Devuelve la hora actual en formato de tiempo Unix.|
|Rlogin|513/TCP|Protocolo de inicio de sesión remoto, utilizado para acceder a una computadora remota de forma insegura.|
|Rshell|514/TCP|Protocolo de shell remoto, utilizado para ejecutar comandos en una computadora remota de forma insegura.|
|Domain Name|53/UDP|Sistema de nombres de dominio, utilizado para resolver nombres de dominio a direcciones IP.|
|TACACS|49/TCP|Protocolo de control de acceso de terminal, utilizado para autenticar usuarios en una red.|
|TACACS-DS|49/UDP|Protocolo de servidor de control de acceso de terminal, utilizado para autenticar usuarios en una red.|
|LPD|515/TCP|Protocolo de impresión en red, utilizado para imprimir documentos en una impresora remota.|
|Kerberos|88/UDP|Sistema de autenticación de red, utilizado para autenticar usuarios en una red.|
|SunRPC|111/TCP y 111/UDP|Protocolo de procedimiento remoto, utilizado para llamar a procedimientos remotos en una computadora remota.|
|NT Domain SMB|135/TCP|Protocolo de intercambio de archivos y servicios de impresión, utilizado para autenticar usuarios en una red de Windows.|
|NetBIOS-NS|137/UDP|Protocolo de nombres de red, utilizado para resolver nombres de NetBIOS a direcciones IP.|
|NetBIOS-DGM|138/UDP|Protocolo de grupo de trabajo de red, utilizado para comunicarse con grupos de trabajo en una red de Windows.|
|NetBIOS-SSN|139/TCP|Protocolo de sesión de red, utilizado para establecer sesiones de red en una red de Windows.|
|LDAP|389/TCP|Protocolo de directorio ligero de acceso, utilizado para acceder a directorios de usuarios y recursos en una red.|
|SNMP|161/UDP|Protocolo de administración de redes simple, utilizado para administrar dispositivos de red.|
|NTP|123/UDP|Protocolo de tiempo de red, utilizado para sincronizar la hora entre computadoras.|
|NNTP|119/TCP|Protocolo de noticias de red, utilizado para leer y enviar noticias a través de una red.|
|IMAP4|143/TCP|Protocolo de intercambio de correo, utilizado para acceder a correos electrónicos en una computadora remota.|
|HTTP|80/TCP|Protocolo de transferencia de hipertexto, utilizado para transferir páginas web entre un cliente y un servidor.|
|HTTPS|443/TCP|Protocolo de transferencia de hipertexto seguro, utilizado para transferir páginas web entre un cliente y un servidor de forma segura.|
|SIP|5060/UDP|Protocolo de inicio de sesión, utilizado para iniciar, mantener y terminar sesiones en redes IP.|
|LDP|646/TCP|Protocolo de enlace de diseño, utilizado para configurar enlaces entre dispositivos de red.|
|Finger|79/TCP|Protocolo de búsqueda de información de usuario, utilizado para obtener información de usuarios en una red.|
|MIME|160/TCP|Multipurpose Internet Mail Extensions, utilizado para describir datos de correo electrónico en Internet.|
|DNSIX|195/TCP|Protocolo de actualización de zona DNS, utilizado para actualizar zonas de búsqueda de nombres de dominio en DNS.|
|FTP-EXT|21/TCP|Extensiones al protocolo de transferencia de archivos, utilizado para realizar tareas adicionales en un servidor FTP.|
|CIFS|445/TCP|Sistema de archivos comunes de Internet, utilizado para acceder a recursos compartidos en una red.|
|NetView|517/UDP|Protocolo de vista de red, utilizado para visualizar y monitorear la configuración de una red.|
|SQL*NET|1521/TCP|Protocolo de red para Oracle, utilizado para conectar aplicaciones a bases de datos Oracle.|
|BGP|179/TCP|Protocolo de enrutamiento entre pares, utilizado para intercambiar información de enrutamiento entre routers.|
|HTTP Proxy|8080/TCP|Protocolo de proxy HTTP, utilizado para reenviar solicitudes HTTP a través de un servidor proxy.|
|X11|6000/TCP|Protocolo de intercambio de imágenes y sonido, utilizado para visualizar imágenes y sonido en una computadora remota.|
|DCE-RPC|135/TCP|Protocolo de intercambio de llamadas remotas de comunicaciones distribuidas, utilizado para comunicarse entre sistemas en una red.|
|JAVA-RMI|1099/TCP|Protocolo de transferencia de recursos de máquina Java, utilizado para acceder a recursos de aplicaciones Java en una computadora remota.|
|Real Time Streaming Protocol (RTSP)|554/TCP|Protocolo de streaming en tiempo real, utilizado para controlar la transmisión de datos en tiempo real en una red.|
|HTTP-MGMT|8000/TCP|Protocolo de administración de HTTP, utilizado para administrar servidores web y proxy HTTP.|
|LDAPS|636/TCP|Protocolo de servicios de directorio ligero de acceso, utilizado para acceder a directorios de usuarios y recursos en una red de forma segura.|
|L2TP|1701/UDP|Protocolo de tunelización de enlace de nivel 2, utilizado para encapsular paquetes IP dentro de paquetes de datagramas de red|
