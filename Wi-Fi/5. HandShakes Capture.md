```
Airodump-ng --essid <nombre red> -w <nombre captura> wlan0mon
```

# Deauth Global


***ES IGUAL QUE EL DIRIGIDO, PERO MODIFICANDO LA MAC DEL CLIENTE ASOCIADO POR LA MAC BROADCAST "FF:FF:FF:FF:FF:FF".***

**CON LO QUE TODOS LOS DISPOSITIVOS ASOCIADOS SE DESCONECTARÍAN. PUDIENDO ASÍ CONSEGUIR DIVERSOS HANDSHAKES.**

# Falsa Autenticación

***ASOCIA 1 CLIENTE (VALIDO SOLO PROTOCOLO WEP)***

```
Aireplay-ng -1 0 -e Test -h <mac falsa> wlan0mon
```

**SATURA LA RED ASOCIANDO MULTIPLES CLIENTES (DOS TEMPORAL)**

```
Mdk3 wlan0mon a -a <MAC AP>
```
