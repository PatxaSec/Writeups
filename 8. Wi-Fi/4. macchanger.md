
***HACER CON LA TARJETA DE RED DADA DE BAJA.***


**Dar de baja la tarjeta:**

```
ifconfig wlan0mon down
```

*MAC escogida manualmente*

```
Macchanger -l | grep "NATIONAL SECURITY AGENCY"

Macchanger --mac="00:20:91:da:ab:17" wlan0mon
```

*MAC aleatoria*

```
Macchanger -a wlan0mon
```

*Recuperar MAC original*
```
Macchanger -p wlan0mon
```

**DAR DE ALTA TARJETA: **

ifconfig wlan0mon up
